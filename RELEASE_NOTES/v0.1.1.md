# MCPAP Vendor Map — Patch v0.1.1

Release date: 2025-10-24
QA: Manual QA completed by user

## Highlights

- New UI: Contact Quick Actions
  - Email quick-copy: click the document icon to copy the vendor email to clipboard. (Toast feedback added)
  - Website quick-open: click the link icon to open the vendor website in a new tab.
  - Email and Website fields now expanded for readability.

## Visual & UX

- New contextual toast notification: "Email copied to clipboard" appears above the copy icon (2000ms).
- Inline contact controls now use reliable SVG icons (no emoji fallbacks).
- Viewer privacy respected: email visibility follows vendor privacy flags; admins retain full access.

## Badges & Overlays

- Returning Vendor badge: refined visuals (ribbon + triangular arrowhead), solid fill, consistent color and zoom behavior.
- Event Staff badge: new ribbon + person glyph, enlarged and centered on both map overlays and panel.
- Panel badge stack: reserved badge column added; staff badge positions above returning badge when present.

## Fixes & Polish

- Copy-to-clipboard robust fallback added (Clipboard API + execCommand).
- Toast repositioning adjusted to avoid overlapping UI elements across zoom and display scales.
- Logo-badge image alignment fixed (object-position:center).
- Booth label and badge anchoring improved to use visual-center helpers.

## Balance & Quality-of-life

- Contact links now concise and clickable; chain/site icon behavior refined for viewers.
- Map and panel badges hide/show in sync with logo badge zoom thresholds.
- Small styling refinements for mobile touch targets and panel layout (further mobile work planned).

## Known Issues

- Full mobile UI audit pending — mobile compatibility tasks are on the roadmap.
- Some micro-alignment differences may remain on extreme zoom / very small booths — further fine-tuning planned.

## Credits

- Implemented: contact quick actions, clipboard toast, badge updates, and UI wiring.
- QA: manual verification performed by user.
