# MCPAP Vendor Map — Patch v0.4.11

Release date: 2026-04-02

Follow-up to **v0.4.10** (mobile animation reliability).

## Mobile (`/mobile`)

- **Pan / zoom animations**: Replaced **`requestAnimationFrame`** loops with **`setTimeout`**-driven steps in **`slowPanTo`**, **`animateZoomTo`**, and **vendor-list → map** pan. On many mobile browsers (notably Safari), **rAF is throttled until user interaction**, so “View on Map” and list deep-link pans appeared to do nothing until the user moved the map.
- **“View on Map”**: **`runWhenMapReadyForAnimation`** runs the pan after **`resize`** + **`idle`** (with a **500ms** fallback) so the camera animation starts after the detail sheet layout settles.
