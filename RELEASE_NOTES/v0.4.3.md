# MCPAP Vendor Map — Patch v0.4.3

Release date: 2026-03-08

## Production server (Gunicorn)

- **Gunicorn config**: Added `gunicorn.conf.py` for production deployment. Binds to `0.0.0.0:PORT` (from `.env` or 5000), 2 workers, 2 threads, 120s timeout.
- **README**: Run instructions updated with a "Production (Gunicorn)" section; use `gunicorn -c gunicorn.conf.py app:app` on Unix/Linux/WSL or production hosts.
- **Requirements**: `gunicorn==23.0.0` added to `requirements.txt`.
- **Note**: Gunicorn is Unix-only (not supported on Windows); use `python app.py` or `flask run` for local development on Windows.

## Technical notes

- Config loads `PORT` via `python-dotenv` so it matches existing `.env` usage.
- Docstring in `gunicorn.conf.py` documents the run command and Unix-only limitation.
