# MCPAP Vendor Map — Hotfix v0.4.6

Release date: 2026-03-08

## Mobile Event Date dropdown

- **Short date format**: The Event Date selector on mobile now uses a compact numeric format (`5/16/26`, `5/17/26`) so the options fit in the dropdown. Desktop still shows the full date string.

## Railway data persistence

- **Vendor data and uploads persist across redeploys**: When `DATA_DIR` is set (e.g. to a Railway Volume mount path), the app stores `vendors.json` and uploaded logos under that directory so redeploys no longer wipe data.
- **Config**: If `DATA_DIR` is set, `UPLOAD_DIR` is `DATA_DIR/uploads` so one volume holds both.
- **README**: Added step-by-step instructions for adding a Railway Volume and setting `DATA_DIR=/data` so deployments keep vendor data and logos.

## Technical notes

- `scheduled-days.js`: added `dateShort` to day options; mobile viewer uses it for the Event Date select label.
- `app.py`: `UPLOAD_DIR` switches to `DATA_DIR/uploads` when `DATA_DIR` env is set.
